Ejercicio 9: Función Fibonacci Refactorizada

Código Original (ES5):

const fibonacci = function(n) {
    if (n < 3) return 1;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

Código Refactorizado (Arrow Function):

const fibonacci = (n) => {
    if (n < 3) return 1;
    return fibonacci(n - 1) + fibonacci(n - 2);
}
